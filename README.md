# Pong-Spiel in Java

Dieses Projekt ist mein erstes größeres Programmierprojekt, das ich erstellt habe. Es handelt sich um einen klassischen "Pong"-Klon.

Features
* Voll funktionsfähige Ball-Physik (Abprallen an Wänden und Schlägern).
* Steuerung des Schlägers über die Tastatur.
* Hintergrund-Grafik und flüssige Animationen.
* Erstellt in der BlueJ-Umgebung unter Verwendung einer Basis-Bibliothek.
